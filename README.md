# PWM-0-10V-Output

A Esphome tool to control a 0-10V output with the help of a PWM signal.

Hardware : 
 - Esp32 Dev board 
 - PWM to Analog (0-10V) Converter Module
 - 12-24V DC power supply 
 - DS18B20 Temperature sensor
 - 4.7K Resistor
